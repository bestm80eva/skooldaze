44256: Lesson descriptors
When a lesson starts, the routine at 26342 picks up the lesson descriptor from this table and copies it into 32756. Each entry in this table defines the teacher for the period (if any), and the room. The teacher is identified by bits 4-7:
Bits Teacher
0000 MR WACKER
0001 MR ROCKITT
0010 MR WITHIT
0011 MR CREAK
0100 None
The room is identified by bits 0-3:
Bits Room
0001 READING ROOM
0010 MAP ROOM
0011 WHITE ROOM
0100 EXAM ROOM
0101 REVISION LIBRARY
0110 DINNER
0111 PLAYTIME
