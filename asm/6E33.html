6E33: Select a special PLAYTIME occasionally
Used by the routine at F905. Three times out of eight, this routine returns with the identifier of a 'special' playtime (243, 244 or 245) in A; the rest of the time it returns with A holding the PLAYTIME identifier from the main timetable.
Input
A Random number
DE Address of the main timetable entry
Output
6E33 AND $07 A=random number between 238 and 245
6E35 ADD A,$EE
6E37 CP $F3 Return if A is 243, 244, or 245 (one of the 'special' playtimes)
6E39 RET NC
6E3A LD A,(DE) Otherwise return with A holding the PLAYTIME identifier from the main timetable
6E3B RET
